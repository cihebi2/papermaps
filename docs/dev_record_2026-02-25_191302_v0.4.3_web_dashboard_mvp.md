7. 接口契约（如有）
- 接口名称：`GET /api/dashboard`
- 用途：供本地 dashboard 拉取核心状态。
- 输入参数：
  - `recent_runs` / int / 否 / 默认服务启动参数 / 最近运行条数。
- 返回值：
  - JSON 对象：`generated_at`、`db_path`、`counts`、`recent_runs`、`watch_targets`。
- 错误处理：
  - 400：`recent_runs` 非法（非整数或 <=0）。
  - 500：内部异常。
- 边界条件：空库返回计数 0 与空列表，不报错。
- 幂等性：GET 查询接口，幂等。
- 使用示例：`GET /api/dashboard?recent_runs=10`。
