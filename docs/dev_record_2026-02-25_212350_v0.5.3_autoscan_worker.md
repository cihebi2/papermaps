## 7. 接口契约（如有）
- 接口名称：`GET /api/autoscan/config`
- 用途：读取当前自动扫描配置。
- 输入参数：无。
- 返回值：
  - `enabled: bool`
  - `interval_seconds: int (>0)`
  - `lookback_days: int (>0)`
  - `max_pages_per_target: int (>0)`
  - `push_new: bool`
  - `last_run_at: string | null`
- 错误处理：服务异常返回 500。
- 边界条件：无配置时返回默认值（300/30/1 等）。
- 幂等性：是。
- 使用示例：`curl http://127.0.0.1:8765/api/autoscan/config`

- 接口名称：`POST /api/autoscan/config`
- 用途：写入自动扫描配置。
- 输入参数：
  - `enabled / bool / 否 / 当前值 / 是否启用后台自动扫描`
  - `interval_seconds / int / 否 / 当前值 / 扫描周期秒数，>0`
  - `lookback_days / int / 否 / 当前值 / 回看天数，>0`
  - `max_pages_per_target / int / 否 / 当前值 / 每目标最多翻页数，>0`
  - `push_new / bool / 否 / 当前值 / 扫描后是否自动推送新告警`
- 返回值：`{ ok: true, ...config }`
- 错误处理：
  - 400：数值参数非整数或 <=0。
  - 500：服务内部错误。
- 边界条件：
  - 传空对象时保持当前值不变。
  - 非法值拒绝写入并返回 400。
- 幂等性：同一配置重复提交结果一致。
- 使用示例：
  - 输入：`{"enabled":true,"interval_seconds":120,"lookback_days":14,"max_pages_per_target":2,"push_new":false}`
  - 输出：`{"ok":true,"enabled":true,...}`
